# Koppelingen tussen applicatie

Koppelen is vanzelfsprekend omdat alle softwarepakketten wel een of meer koppelingen hebben met andere applicaties. Via deze koppelingen vindt uitwisseling plaats van gegevens en documenten.

#### Faster Forward
Faster Forward biedt voor financiële adviseurs het softwarepakket [Elements](https://www.fasterforward.nl/elements/). Elements heeft veel [koppelingen](https://www.fasterforward.nl/koppelingen/) naar andere applicaties. Via een [online kennisbank](https://support.fasterforward.nl/kennisbank/) geeft Faster Forward uitleg over onder meer de koppelingen.

Hieronder staat een aantal voorbeelden van koppelingen:

De Faster Forward Elements API collectie is set van objectbeschrijvingen die het mogelijk maken om te connecteren met het Faster Forward Elements Platform. 

De API is opgebouwd rondom de volgende technieken en methodieken:

- Architectuur [(REST - Representational state transfer)](http://www.ics.uci.edu/~fielding/pubs/dissertation/rest_arch_style.htm)

- Autorisatie [(OAuth 2.0)](http://oauth.net/2/)

- Response data format [(JSON - JavaScript Object Notation)](https://tools.ietf.org/html/rfc7159)

- Response data document format [(HAL - Hypertext Application Language)](http://stateless.co/hal_specification.html)

- Response data document error format ([vnd.error](https://github.com/blongden/vnd.error))

#### Adviesbox
Met Adviesbox vraagt een adviseur gemakkelijk elektronisch een bankgarantie, taxatie of hypotheek aan bij verschillende aanbieders. Daartoe heeft Adviesbox koppelingen met het HDN of direct met de systemen van de aanbieders. 

Adviesbox heeft onder meer [koppelingen](https://support.adviesbox.nl/portal/nl/kb/intersoftware/koppelingen) met 

- Advieskeuze.nl (review platform waar consumenten de financieel dienstverlener kunnen beoordelen)

- Assistent

- [Duokoop](https://www.duokoop.nl/)

- EDR Credit Check (toetsing kredietwaardigheid)

- Elements

- Elements van Faster Forward

- Finly webtool

- HDN

- Invice

- Kompas

- Nationale Hypotheekbond

- Nationale Notaris

- Nationale Waarborg voor bankgarantie

- NWWI voor taxatieaanvragen

- Rolls

- Serviceproviders

- Slim

- VCN - United Capital voor krediet aanvraag
