# Artikel 992590

Onderstaande tabel laat voorbeelden zien van databronnen die voor consumenten belangrijk kunnen zijn.

| 
**Databron**

 
| 
**Data**

 
 |

| MijnOverheid: Basisregistratie Personen

 
| Naam, geboortedatum, email, woonadres, familie, nationaliteit, paspoortnummer.

 
 |

| MijnOverheid: Kadaster/WOZ

 
| Met gegevens over bestaande woning, WOZ-rapport van de gemeente, kadaster gegevens.

 
 |

| MijnOverheid: MGI Mijn Geregistreerd Inkomen / IBRI

 
| Onder andere voor vaststellen inkomen van zzp’ers.

 
 |

| Belastingdienst: VIA

 
| Vooraf ingevulde aangifte bij de Belastingdienst met bekende informatie over huishouden, inkomen, schulden, bezittingen, huis en hypotheek.

 
 |

| UWV

 
| Biedt 5 jaar informatie over werkgevers, salaris en eventuele uitkeringen.

 
 |

| MijnPensioenOverzicht

 
| Pensioendata beschikbaar via het Pensioenregister.

 
 |

| DUO

 
| Studieschulden.

 
 |

| BKR

 
| Toetsen ten behoeve van kredietwaardigheid en KYC: VIS toets. PEP toets. Sanctie toets, Insolventie toets, BKR indicatie.

 
 |

| Banken (op basis van PSD2)

 
| 
Banktransacties. Actuele saldi van betaal- en spaarrekeningen, creditcards, investeringsrekeningen.

Gecategoriseerd naar soort uitgaven en inkomsten.

 
 |

| BRK

 
| Basisregistratie Kadaster

 
 |

| BRI

 
| Basisregistratie Personen

 
 |

| BRV

 
| Basisregistratie Voertuigen

 
 |
